# 자동매매 백로그 (전략 로드맵)

자동매매 봇의 단계별 개발 계획. 모든 전략은 기존 파이프라인을 재사용한다:
`strategy/` → `orders`(+outbox) → `relay` → 체결엔진 → `executions` → 포트폴리오.

각 항목은 **별도 이슈 → 브랜치 → PR** 사이클로 진행한다.

---

## ✅ 1단계 — 기본 전략 (진행: #40)
- [ ] 이동평균 교차(SMA): 단기/장기 SMA, 골든크로스 매수 / 데드크로스 매도
- [ ] 계정별 자동매매 토글(`accounts.auto_trade`) + 대시보드 ON/OFF
- [ ] 안전장치: 주문당 고정 KRW, 종목당 1포지션, 종목별 쿨다운
- [ ] 공유 주문 헬퍼(`common/order_writer.py`)로 API·봇 주문 경로 통일

## 2단계 — 추가 전략 (추후)
- [ ] **RSI**: 과매도(<30) 매수 / 과매수(>70) 매도 (mean-reversion)
- [ ] **돌파/모멘텀**: 최근 N분 신고가 돌파 매수, 하락 매도
- [ ] **MACD**: 시그널선 교차
- [ ] **볼린저 밴드**: 밴드 이탈/회귀
- [ ] 계정별 전략 선택 + 파라미터 설정 UI

## 3단계 — 리스크/주문 관리 (추후)
- [ ] 손절(stop-loss) / 익절(take-profit)
- [ ] 포지션 사이징(잔고 비율 기반), 최대 노출 한도
- [ ] 슬리피지/수수료를 반영한 신호 필터(과매매 방지)

## 4단계 — 백테스트 & 성과 (추후)
- [ ] ClickHouse 틱/캔들 replay 기반 백테스트 러너 (Kafka offset 되감기 활용)
- [ ] 전략별 성과 지표: 누적수익률, 승률, MDD, 샤프
- [ ] 대시보드 성과 패널
