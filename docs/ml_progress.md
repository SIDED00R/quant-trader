# 주식 ML 파이프라인 — 점진적 개선 연대기

> 각 단계: **무엇을 추가했나 → 왜 → 결과(측정)**. 모든 측정은 동일 규율(purged/embargo walk-forward CV + Rank IC/ICIR + Newey-West 겹침보정 t값 + 롱숏 Sharpe, 시장별 US/KR 분리). 권위 기록은 git 커밋(#143~), 큐레이트 표는 [ml_models_research.md](ml_models_research.md). **수치는 생존편향(현 구성종목 소급) 미보정이라 과대 — 절대값보다 *증분*과 *방향*을 본다.**

## 0. 토대 — 전체 유니버스 검증 (Issue#143 / PR#144)
- **무엇**: KOSPI200+KOSDAQ150+S&P500+NASDAQ100 구성종목 856종목 일봉 7.1년 백필 → 횡단면 모멘텀 walk-forward.
- **왜**: 6·30종목 소표본에선 −33~+108% 요동으로 "기각"처럼 보였음. breadth(종목 폭)가 빠진 조각인지 확인 필요.
- **결과**: 사전확정 lb60/N10 = KR +481%/DSR0.990, US +604%/DSR0.998. 룩백/N 전반 안정(과적합 아님), 슬리피지 20bps 생존. **breadth가 결론을 뒤집음.** 단 생존편향이 핵심 미결.

## 1. ML 피처 토대 (Issue#145)
- **무엇**: OHLCV 파생 피처 ~58개(모멘텀·변동성 레인지추정·유동성·Alpha158 KBar/롤링·기술지표) + Rank IC/ICIR/NW-t 유용성 하니스 + purged/embargo CV.
- **왜**: 논문 전수조사(213피처) 결론 — GKX 최상위 예측변수 대부분이 OHLCV로 계산 가능. 모델보다 피처가 90%.
- **결과(단변량 IC, 참고)**: Amihud 유동성이 양 시장 최강(US/KR NW_t~3.1). KR=교과서 이상현상(저변동성·복권·단기반전), US=강세장/생존편향 왜곡. → KR/US 분리 확정.

## 2. GBDT 베이스라인 (게이트)
- **무엇**: LightGBM, 시드앙상블, 라벨=횡단면 z-score.
- **회귀 → lambdarank 손실 교체**: 평가(Rank IC)와 학습목표 정렬.
- **결과(US, OOF 5시드6fold)**: 회귀 2.06%/NW_t1.6 → **lambdarank 3.09%/NW_t1.8**. KR은 0.99%/0.67%로 약함(OHLCV만으론 OOS 신호 부족 — KR은 외국인수급 필요 시사).

## 3. 데이터-우선 전환 + 외부데이터 ablation (핵심)
- **무엇/왜**: "모델 검증과 무관하게 데이터를 다 수집·저장"(재사용 자산·PIT 소실 방지). `batch/data/` 멱등 수집기로 영구 저장: SEC EDGAR 펀더멘털·FRED 매크로·US PIT멤버십·13F 기관보유·SEC SIC 섹터. **각 외부데이터를 ablation으로 유용성 측정 후 선별.**
- **결과(US lambdarank, 누적)**:
  | 추가 | 이유 | Rank IC | NW_t | 판정 |
  |---|---|---|---|---|
  | OHLCV 전용 | 베이스 | 3.09% | 1.8 | 기준 |
  | **+ 펀더멘털**(EDGAR 공시) | GKX 가치/규모/수익성 팩터 | **3.41%** | **2.1** | 채택(유의성 첫 돌파) |
  | **+ 13F**(기관 보유·QoQ) | 기관 누적/분산 신호 | **3.56%** | **2.2** | 채택 |
  | **+ 섹터**(산업모멘텀 indmom) | GKX top-7, 섹터중립 | **3.78%** | **2.3** | 채택(현 챔피언) |
  | + 매크로(금리·VIX·환율) | 레짐 | 2.05% | 1.5 | **제외(시장수준→과적합)** |
- **교훈**: **검증된 외부데이터(펀더멘털·13F·섹터)는 누적 개선**(NW_t 1.8→2.3). **매크로(시장수준)는 무지성 추가 시 과적합** → ablation 선별 필수.
- **KR**: 외국인수급·공매도·KR펀더멘털 수집·검증 완료 → §7.

## 4. DL 비교 (완료, Issue#148) — 트리 압승, DL 미달
- **무엇**: GBDT(lambdarank) vs MLP vs GRU(raw OHLCV 5채널×60일) head-to-head. 동일 데이터·purged CV·자체완결 스크립트(`batch/ml/colab_train.py` — 보류 확정 후 #196에서 코드 삭제, git 히스토리 보존)로 같은 파이프라인 비교.
- **왜**: 시퀀스 DL이 트리(공학피처)와 직교 정보원일 수 있나? 단 tabular는 GBDT가 강하다는 prior.
- **결과(최종, 동일 파이프라인)**:
  | 모델 | 입력 | Rank IC | NW_t | LS_Sharpe | 판정 |
  |---|---|---|---|---|---|
  | **GBDT**(lambdarank) | 공학피처 70 | **3.40%** | 2.2 | 1.20 | 챔피언 |
  | MLP | 같은 70 | 1.22% | 1.5 | 0.15 | 미달 |
  | GRU | raw OHLCV 5ch×60일 | 0.27% | 0.3 | 0.38 | 사실상 0 |
- **인프라**: 무료 Colab GPU는 한도소진·세션회수로 실패 → **Kaggle 커밋세션(P100)** 으로 GRU 풀학습(5시드×6fold×40ep, 112분). P100=sm_60이라 기본 torch(cu128) 미지원 → 커널서 `torch==2.4.1+cu118` 재설치.
- **로컬 GPU 확보(2026-07)**: 로컬 PC에 8GB VRAM GPU 사용 가능 — GBDT 챔피언은 데이터 규모(~344종목×7년, ~50만 행)상 CPU 유지(변경 없음). 향후 DL 재실험 시 Kaggle 커밋세션 대신 로컬 GPU 사용.
- **교훈**: GRU는 **GPU 풀학습으로도 엣지 없음**(축소 CPU 0.37%와 일치 → 과소적합 아닌 **구조적 미달**). 정형 횡단면 금융데이터는 GBDT가 DL 압도(문헌 일치). GRU는 입력 핸디캡(펀더/13F 없이 raw만)도 일부. **DL 보류 — 다음 개선은 피처·레짐·생존편향 통제 방향.**

## 5. GBDT 하이퍼파라미터 튜닝 (누설 없음, Issue#148)
- **무엇/왜**: 고정 HP가 최적인지 검증. **시간분리**(앞 45% 날짜=튜닝창 탐색 / 뒤 55%=평가창은 탐색 미관여)로 누설 차단.
- **결과**: 24조합 탐색 → held-out 평가창서 default **6.62%** → tuned **6.82%**(소폭). best = 더 **보수적**(num_leaves15·min_child300·feature_fraction0.6) → 기존 고정값이 이미 거의 최적. 대규모 탐색은 효용 낮음(이득 작고 다중검정 과적합 위험).
- **부수 발견**: 평가창(최근 55%) IC가 전체기간(3.40%)보다 훨씬 높음 → 엣지가 **최근 구간에 집중**(레짐효과·생존편향 주의).

## 6. 코드 품질 (Issue#146, 머지됨)

## 7. KR 외부데이터 수집 + ablation (Issue#150/152/154/156)
- **무엇/왜**: KR의 OHLCV-only ML이 OOS 약함(§ baseline)이라, "진짜 unlock은 외국인수급·공매도·펀더멘털" 가설을 검증. KRX(pykrx 로그인)·DART(키) 자격증명 확보 후 수집기 구축.
- **수집기**: `batch/data/krx.py`(수급·공매도·외국인보유 → stock_investor_flow/short/foreign_holding), `kr_index_membership.py`(KOSPI200/KOSDAQ150 PIT), `kr_fundamentals.py`(DART → fundamentals_quarterly, source='DART', US와 동일 규약 plug-in). 피처: `batch/features/kr_microstructure.py`(누설차단 8종) + `dataset.py` KR 경로 배선.
- **단변량 Rank IC(fwd21d, 2024–2026, 344종목)**: 공매도잔고비율 **−7.4%(NW_t −6.3)**·외국인보유율 +4.1%·외국인순매수20 +3.25%·외국인보유Δ +3.16%·연기금/기관 순매수 −3.9%/−3.5%(역방향). 단 full-sample·공매도금지 레짐 영향 → OOS로 재검증. ⚠ 기관 순매수 단변량은 pension 이중계산(#158) 영향 가능 — 권위는 아래 다변량 ablation.
- **OOS ablation(purged WF, lambdarank, seed5/fold6, 7y/1474일). pension 이중계산 수정 후 재검증(#158, 현 데이터 기준)**:
  | 모델 | Rank IC(7y) | ICIR | NW_t | LS_Sharpe | LS_mean |
  |---|---|---|---|---|---|
  | OHLCV 전용 | 0.23% | 0.02 | 0.2 | 1.03 | 1.91% |
  | **+ DART 펀더멘털** | **1.17%** | 0.09 | **1.0** | **1.40** | 2.52% |
  | + KRX 미시구조(수정) | 1.15% | 0.09 | 0.9 | 1.39 | 2.60% |
- **교훈(수정 후 결론 변경·중요)**: `kr_microstructure`의 기관 순매수 inst_net이 KRX 기관합계(연기금 포함)에 pension을 재합산 → **연기금 이중계산**(#158; 적재 7년 전체가 벌크소스라 전구간 오염). 수정 후 재검증하니 **미시구조 증분 소멸**(+DART 1.17% → +미시 1.15%, Δ−0.02%). 기존 "미시 unlock(0.52%→1.21%)"은 **pension 아티팩트 + §7 당시 DART 부분적재(06-29 11:46 UTC 재적재 전)**의 합작이었음. **진짜 KR 동력 = DART 펀더멘털**(OHLCV 0.23% → +DART **1.17%**, NW_t 0.2→1.0, LS_Sharpe 1.03→1.40; 중요도 상위 fund_mktcap·psr·roe·roa·per·pbr 독식). 미시 중 외국인보유율(kr_fh_ratio)만 트리에 미미 사용·OOS 증분 0. **판정: KR 외부데이터 가치는 DART 펀더멘털에 있음. KRX 미시구조(수급·공매도·외국인보유)는 현 구성서 증분 0 → 채택 보류.** 단 NW_t~1.0은 US(2.3) 대비 약함 — KR 구조적 약신호 유지. 공매도 단변량 −7.4%의 강함은 full-sample·공매도금지(2020-03~2021-05, 2023-11~2025-03) 레짐 효과(별도 검증=task 2).
- **공매도 레짐 분리 검증(task 2, #160; 4렌즈 적대적 감사 + 실측)**: §7 단변량 −7.4%(2024-26 창)가 공매도 금지레짐(2020-03~2021-05, 2023-11~2025-03) 아티팩트인지 검증. 신호일 기준 레짐 분리 단변량 Rank IC(공매도잔고비율, fwd21d):
  | 구간 | meanIC | NW_t | days |
  |---|---|---|---|
  | full(7y) | −4.34% | −4.9 | 1724 |
  | normal 전체 | −3.21% | −2.8 | 1105 |
  | W3(2025-03~ 완전해제) | **−6.28%** | **−3.0** | 282 |
  | W2 member만(실제 shortable) | −2.05% | −0.9 | 619 |
  | **보정 normal**(부분해제 금지잔존 제외) | −2.28% | −1.5 | 1105 |
  - **판정: "순수 금지레짐 아티팩트" 기각 — 신호는 진짜.** 완전해제 W3에서 −6.28%/t−3.0로 가장 강하고, 경계누설 제거(−3.61%/t−3.1)·진입 1일지연(−3.20%/t−2.8)에도 불변(누설·타이밍 아티팩트 아님). **그러나 약·비정상**: 2021-05 부분해제(KOSPI200/KOSDAQ150만 허용)를 보정해 W2 금지잔존 비멤버를 빼면 normal이 −2.28%/t−1.48(거래량비율 −1.17%/t−1.45)로 **유의성 소멸**. W3·2022약세장(−5.35%/t−2.5) 집중, W1(2019) 부호반전(+2.54%). 헤드라인 −7.4%는 금지레짐+부분해제 혼입 ~2-3배 증폭치. **DART-value 잔차엔 독립 알파 잔존**(잔고 −2.12%/t−2.2, 거래량 −2.91%/t−3.1)이나 **전체 OHLCV+DART 모델선 흡수**(#158 +micro≈0; OHLCV 비유동성·변동성이 대체). **최종: 공매도 신호는 실재하나 OHLCV+DART 챔피언에 채택할 증분 없음(보류 확정). 운용 가정 수치는 full −7.4% 아닌 normal 기준.** (적대적 감사: 누설·통계·경제 렌즈 지지, 레짐교란 렌즈가 부분해제 정의결함 적발→실측 반영. 통계렌즈 독립재현 −3.21%/t−2.76·Bonferroni p0.047·block-bootstrap CI[−5.5,−0.9].)
- **운영 메모**: per-symbol 수집기는 장기범위 느림(공매도 8년 ~3시간) → **by-date 전종목 함수**(`get_*_by_ticker`, `get_market_net_purchases_of_equities_by_ticker`)로 벌크 리팩터 권장(검증 적재는 이 방식으로 수행). pykrx는 **import 시점 KRX_ID/PW 로그인** → .env를 import 전 로드. DART는 thstrm=분기3개월·연간12개월(Q4=연간−Q3누적).
- **챔피언 확정(task3a)**: KR 챔피언 = **OHLCV + DART 펀더멘털**(macro·미시구조 제외). 7y purged-WF: OHLCV-only(no macro) 0.67% → **+DART 1.34%/NW_t 1.1/LS_Sharpe 1.42**. **macro 추가 시 1.17%로 악화**(US와 동일 — 시장수준 과적합), 미시구조 증분 0(#158). 재현: `python -m batch.ml.baseline_lgbm KR`(플래그 없음 = 챔피언: macro·kr-micro 기본 제외). 라이브 스코어러 `batch/ml/stock_score.py`가 이 설정.
- **생존편향 소스 확보(task3b)**: 과거 "상폐 가격 무료소스 없음" **반증** — `FinanceDataReader('KRX-DELISTING')`(상폐 4,146종목 + 폐지일·사유 + 종목별 OHLCV)·`marcap`(1995~ by-date 스냅샷이 상폐종목 보존=생존편향-free)·pykrx `adjusted=False`(KRX 원시가). 보정안: FDR(상폐 라벨)+marcap(벌크 PIT 가격) → 별도 수집 트랙. 절대 IC 과대(생존편향)의 본질적 해결책.
- **실모의 거래 배선(PR2)**: 챔피언 스코어러 → KIS 모의주문(`kis_order`) → 체결확인(잔고-diff; KIS 모의는 일별체결조회 미지원) 배선·실측(삼성 등 체결, 주문번호 0000014493). 대시보드 주식 탭(PR1)에 표시. KIS 호출은 `rate_limit`(5/s)+토큰 파일캐시. 운영화(동일가중 사이징·주간 스케줄·US 해외주문) 후속.
